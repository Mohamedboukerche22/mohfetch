# how to install 
run this on your terminal 
```sh
git clone https://github.com/Mohamedboukerche22/mohfetch.git && cd mohfetch && chmod +x mohfetch && sudo mv mohfetch ~/.local/bin/mohfetch
```
and voila it's done 
test it by typping in your terminal `mohfetch`
